# Network-Protocol-Configuration-Security-Lab-Using-Cisco-Packet-Tracer
Configured 10+ protocols by implementing server, switches, and laptops using Cisco Packet Tracer.


## Overview
Designed and implemented an enterprise LAN environment using Cisco Packet Tracer, configuring multiple networking and security protocols for secure communication and network management.

## Tools Used
* Cisco Packet Tracer

## Protocols Implemented
* ARP
* ICMP
* DHCP
* DNS
* HTTP
* HTTPS
* FTP
* SSH
* Telnet
* VLAN
* STP
* Port Security

## Network Components
* Cisco 2960 Switches
* Server
* End Devices (PCs)

## Key Tasks Performed
* Configured static and dynamic IP addressing
* Implemented DNS and web services
* Enabled FTP file transfer
* Configured SSH and Telnet remote management
* Created VLAN segmentation
* Verified STP operation
* Implemented switch port security

## Outcome
Successfully created a secure enterprise network simulation with multiple protocols, service validation, and security hardening.
